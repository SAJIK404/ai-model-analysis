AI Model Benchmark Portfolio

Un sitio técnico que presenta métricas de valor, precio y rendimiento para modelos de IA contemporáneos.

Los resultados se actualizan semanalmente desde GitHub Actions y se consumen desde un JSON estático.

Visión general del proyecto

Este análisis combina benchmarks de MMLU, HumanEval, MATH y GPQA con pricing histórico para generar un ranking de valor por dólar. La evaluación usa una métrica compuesta de score promedio / costo promedio por 1M tokens.
